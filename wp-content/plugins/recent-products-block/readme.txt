Display WooCommerce Recent Products

== Description ==

Recent Products Block for WooCommerce is a block plugin by which you can showcase your WooCommerce Recently added product in sidebar, or anywhere in the block editor area. 


**[Demos](https://bblockswp.com/demo/woocommerce-recent-products/)**


= Features =
- **Query**: Display product as your query. You can show products by stock filter, category filter etc.
- **Layout**: Set different columns in different devices for responsive. Also, custom column and row gap.
- **Element Show/Hide**: Show or hide product elements, such as product image, title, rating, price, add to cart button.


= How to use =
- First install the WooCommerce Recent Products plugin
- Add the WooCommerce Recent Products block from the block category called "Widgets" in Gutenberg editor.
- You can change block settings from the right side settings sidebar.
- Enjoy!

* For installation help click on Installation Tab

== Installation ==

= From Gutenberg Editor: =
1. Go to the WordPress Block/Gutenberg Editor
2. Search For **WooCommerce Recent Products**
3. Click on the **WooCommerce Recent Products** to add the block

= Download & Upload: =
1. Download the **WooCommerce Recent Products** plugin (*.zip file*)
2. In your admin area, go to the Plugins menu and click on **Add New**
3. Click on **Upload Plugin** and choose the **`recent-products-block.zip`** file and click on **Install Now**
4. Activate the plugin and Enjoy!

= Manually: =
1. Download and upload the **WooCommerce Recent Products** plugin to the **`/wp-content/plugins/`** directory
2. Activate the plugin through the Plugins menu in WordPress
